# Pesquisa-Ação

Esta pasta documenta os ciclos de pesquisa-ação realizados durante o desenvolvimento da tese.

## Metodologia

A pesquisa-ação é uma metodologia de pesquisa qualitativa que combina teoria e prática através de ciclos iterativos de:

1. **Planejamento** - Identificação do problema e planejamento da ação
2. **Ação** - Implementação das mudanças planejadas
3. **Observação** - Coleta de dados sobre os efeitos da ação
4. **Reflexão** - Análise dos resultados e aprendizados

## Diários de Pesquisa

A pasta `diarios/` contém registros contínuos do processo de pesquisa, incluindo:
- Observações diárias
- Reflexões sobre o processo
- Desafios encontrados
- Insights e aprendizados
- Decisões metodológicas

## Organização de Cada Ciclo

Cada pasta de ciclo deve conter:

```
ciclo-XX/
├── README.md              # Descrição do ciclo
├── planejamento.md        # Plano de ação
├── dados/                 # Dados coletados
├── observacoes/           # Observações do ciclo
├── analises/              # Análises realizadas
└── reflexoes.md           # Reflexões e aprendizados
```

## Contexto da Pesquisa

Esta pesquisa-ação documenta o desenvolvimento do **Sistema Tocantins Integrado**, uma plataforma de inteligência territorial que utiliza IA para apoiar a tomada de decisão em políticas públicas no estado do Tocantins.

O processo envolve:
- Desenvolvimento de agentes de IA especializados
- Integração com bases de dados governamentais
- Testes e validação com potenciais usuários
- Documentação de desafios técnicos e organizacionais
- Análise de viabilidade de integração na administração pública
