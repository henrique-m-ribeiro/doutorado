# Disciplinas

Esta pasta organiza todos os materiais relacionados às disciplinas cursadas durante o doutorado.

## Organização

Cada disciplina deve ter sua própria pasta com a seguinte estrutura:

```
[nome-da-disciplina]/
├── trabalhos/      # Trabalhos, atividades e projetos da disciplina
├── referencias/    # Textos de referência, artigos e livros
├── slides/         # Slides de aulas e apresentações
└── notas/          # Anotações pessoais e resumos
```

## Como adicionar uma nova disciplina

1. Crie uma pasta com o nome da disciplina (use hífens para separar palavras)
2. Crie as subpastas: `trabalhos/`, `referencias/`, `slides/`, `notas/`
3. Adicione um arquivo `README.md` na pasta da disciplina com:
   - Nome completo da disciplina
   - Professor(a) responsável
   - Período/semestre
   - Ementa resumida
   - Objetivos de aprendizagem

## Exemplo de estrutura

```
disciplinas/
├── politicas-publicas-territoriais/
│   ├── trabalhos/
│   ├── referencias/
│   ├── slides/
│   ├── notas/
│   └── README.md
├── metodologia-pesquisa-acao/
│   ├── trabalhos/
│   ├── referencias/
│   ├── slides/
│   ├── notas/
│   └── README.md
└── README.md (este arquivo)
```

## Dicas de organização

- Use nomes descritivos para os arquivos
- Mantenha as referências em formato PDF quando possível
- Organize notas por tópico ou data
- Numere trabalhos cronologicamente (ex: `trabalho-01-revisao-literatura.pdf`)
