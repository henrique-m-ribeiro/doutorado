## Pergunta Central de Pesquisa

> "Como um sistema de inteligencia territorial baseado em IA, co-criado atraves de pesquisa-acao, pode transformar a formulacao de politicas publicas e a governanca para desenvolvimento regional no Brasil, e quais sao as implicacoes desse processo para a colaboracao humano-IA e para a gestao publica?"
