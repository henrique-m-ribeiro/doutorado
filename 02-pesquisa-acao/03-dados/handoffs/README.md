# Handoffs de Sessoes

Documentos de transicao entre sessoes de trabalho com agentes IA, seguindo o framework IA-Collab-OS.

## O que sao Handoffs

Handoffs sao documentos estruturados que registram:
- Resumo executivo da sessao
- Caminho percorrido (decisoes, acoes, resultados)
- Decisoes arquiteturais (ADRs)
- Estado do projeto ao final da sessao
- Proximos passos e questoes abertas
