# Diarios de Sessoes de Pesquisa-Acao

Diretorio central para registro reflexivo de todas as sessoes de trabalho do doutorado.

## Convencao de Nomenclatura

```
YYYY-MM-DD-sessao-NNN.md
```

- `YYYY-MM-DD`: Data da sessao
- `NNN`: Numero sequencial da sessao (001, 002, ...)
- Continuacoes: `YYYY-MM-DD-sessao-NNN-continuacao.md`

## Instrucoes para Novos Diarios

1. Usar o template: [TEMPLATE-DIARIO.md](./TEMPLATE-DIARIO.md)
2. Seguir a convencao de nomenclatura acima
3. Preencher TODAS as secoes obrigatorias do template
4. Vincular ao ciclo PA correspondente (ver `01-ciclos/`)
5. Declarar explicitamente a questao de pesquisa enderecada
